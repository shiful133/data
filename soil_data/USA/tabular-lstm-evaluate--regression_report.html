What do these values mean?

The row count is the number of records or lines in the real data and synthetic data that train the machine learning models. The column count is the number of fields in each dataset. The rows for real and synthetic training data may be downsampled to a maximum of 5000 rows in order to provide a fast and accurate machine learning score comparison while ensuring that your Gretel credits are going toward synthetic model training.

The Lines Duplicated value indicates any possible duplications between synthetic and real data.
